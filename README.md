## Simple QR Code Generator

This website is a QR Code Generator, that can generate...well, a QR code for any valid link entered.

# Features

1. A colored QR Code (Instead of the usual black and white QR codes, I found a way to create a colorful QR code with blue and purple hues)
2. Copy feature (The QR Code can be copied by the user through clicking a simple button)
